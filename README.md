Stock Window is a holiday project exploring the realm of predictive time-series stock forecasting and comparison.
The applications UI is powered through the streamlit python library while time-series forecasting is supported through yfinance, matplot, pandas, pmdarima and numpy.

SQL database creation and querying has been incorporated which can be updated by the user.

To run:
Open up a venv and run

pip install requirements.txt
streamlit run widget.py
